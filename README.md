# Playwright Web Automation Practice (Python)

This repository contains my practice work on web automation using **Playwright with Python**. The project is created to learn and experiment with browser automation techniques used in modern QA testing.

The scripts automate common user interactions such as navigating web pages, filling forms, clicking elements, and validating UI behavior.

---

## 🚀 Technologies Used

- Python
- Playwright
- PyCharm
- CSS Selectors / Locators

---

## 📂 Project Purpose

The goal of this repository is to:

- Learn browser automation using Playwright
- Practice locating web elements
- Automate common UI interactions
- Improve QA automation skills

## ⚙️ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/your-repository-name.git
cd your-repository-name
 ```

### 2. Install Dependencies

```bash
pip install playwright
```

### 3. Install Playwright Browsers

```bash
playwright install
```

### 4. Run the Script

```bash
python filename.py
```
